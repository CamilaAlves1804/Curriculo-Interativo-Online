Livraria

O projeto Livraria é um script Python que gerencia uma livraria virtual. Ele permite ao usuário cadastrar, consultar e remover livros do acervo. As funcionalidades incluem o cadastro de livros com informações como ID, nome, autor e editora, a consulta de livros por ID ou autor, e a remoção de livros do sistema. Este script é uma solução simples para o gerenciamento de um pequeno acervo de livros.

Gráfica

A Calculadora de Serviços de Copiadora é um script Python que calcula o custo total dos serviços oferecidos por uma copiadora. O usuário pode escolher entre diferentes tipos de serviços (digitalização, impressão colorida, impressão preto e branco, fotocópia), informar o número de páginas e selecionar serviços adicionais. O script aplica descontos progressivos com base no número de páginas e calcula o custo total, incluindo quaisquer serviços adicionais selecionados.

Site Currículo

O Currículo Online é um projeto de site pessoal desenvolvido em HTML e CSS, apresentando um currículo interativo. O site é estruturado com seções para Dados Pessoais, Formação Acadêmica, e Habilidades. O layout é responsivo e adaptável a diferentes tamanhos de tela, utilizando práticas modernas de design como containers com bordas, imagens de navegação, e áreas de texto para scripts de exemplo e demonstrações em GIF. O portfólio inclui também uma seção para scripts e exemplos práticos.
